### Color Customization
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #8B7355;       /* Warm brown */
    --secondary-color: #A0956B;     /* Olive brown */
    --accent-color: #6B8E5A;        /* Sage green */
    --text-primary: #2D2A26;        /* Dark brown */
    --bg-primary: #FAF9F7;          /* Warm white */
    /* ... more colors */
}
```
